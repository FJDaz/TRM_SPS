# TRM Backend

## Structure

```
Backend/
├── Notebooks/    # Notebooks Jupyter pour expérimentation et développement
└── Prod/         # Code de production (FastAPI, modules Python)
```

## Composants Principaux

- **Router** : Détermination philosophe/corpus
- **RAG Retriever** : Récupération passages pertinents
- **BERT Encoder** : Condensation mémoire conversationnelle
- **Mistral Generator** : Génération réponse
- **Mini-store** : Détection motifs récurrents

Voir `docs/guides/PRD_TRM_ENRICHI.md` pour détails complets.
